# Niimbot D110 Thermal Printer Setup ✅

## What's Been Configured

Your sticker generator is now fully compatible with the **Niimbot D110 thermal printer**!

## Supported Sticker Sizes

The following Niimbot D110 compatible sizes are now available:

1. **12 × 40 mm** - Standard Niimbot D110 size
2. **15 × 50 mm** - Extended Niimbot D110 size

## How to Use

### Option 1: Print from Sticker Generator Page

1. Go to **Sticker Generator** page
2. Select products you want to print stickers for
3. Click one of the Niimbot D110 buttons:
   - **12×40 (D110)** - For 12×40mm stickers
   - **15×50 (D110)** - For 15×50mm stickers
4. Print the page using your browser's print dialog

### Option 2: Generate ZPL File (Recommended for Direct Printing)

1. Click the **"Generate Stickers"** button (QR code icon)
2. Select sticker size:
   - **12 × 40 mm (Niimbot D110)**
   - **15 × 50 mm (Niimbot D110)**
3. Choose output format: **ZPL (Thermal Printer)**
4. Click **Generate**
5. The `.zpl` file will download automatically

### Using the ZPL File with Niimbot D110

**Method 1: Via Niimbot App**
1. Transfer the `.zpl` file to your phone/tablet
2. Open the Niimbot app
3. Import the ZPL file
4. Print directly to your Niimbot D110 printer

**Method 2: Direct Printing (if supported)**
- Some thermal printer software can print ZPL files directly
- The ZPL format is optimized for 203 DPI (Niimbot D110 resolution)

## Technical Specifications

- **Printer Resolution**: 203 DPI (optimized)
- **Supported Formats**: PDF (for preview/printing) and ZPL (for thermal printer)
- **Barcode Format**: CODE128 (compatible with Niimbot D110)
- **Label Width**: Up to 15mm (Niimbot D110 maximum)

## Printer Setup Tips

1. **Load labels correctly** - Printable side facing down
2. **Calibration** - Printer auto-calibrates on startup
3. **Label material** - Use thermal labels for best results
4. **Bluetooth connection** - Connect via Niimbot app on iOS/Android

## Troubleshooting

**If stickers don't print correctly:**
- Ensure you're using the correct size (12×40mm or 15×50mm)
- Check that labels are loaded properly in the printer
- Verify Bluetooth connection is active
- Try using the ZPL format instead of PDF for direct printing
